# Countdown Timer

A real-time countdown timer that supports 
seconds, minutes and hours.

## Features
- Choose between seconds, minutes or hours
- Counts down in real time with one unit pause between each tick
- Loop to run multiple countdowns in one session
- Input validation for unit selection

## How to run
1. Compile: g++ countdown.cpp -o countdown
2. Run: ./countdown

## What I learned
chrono and thread libraries, real-time program 
behavior and if/else if chains.
